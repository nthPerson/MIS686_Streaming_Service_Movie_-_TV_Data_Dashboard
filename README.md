# Streaming Service Movie & TV Data Dashboard

![Movie Monkies Logo](streamlit/movie_monkies_logo.png)

Link to app website: [Streaming Platform Content Analytics Dashboard](https://streaming-media-intelligence-dashboard.streamlit.app/)

Streamlit-based analytics comparing Netflix, Amazon Prime Video, Hulu, and Disney+ catalogs. The app combines ETL, a normalized MySQL schema, and role-aware dashboards to answer eight analytical questions about catalog size, mix, geography, genre concentration, and maturity profiles.

- Set up database (see [APP_SETUP_README.md](APP_SETUP_README.md))
